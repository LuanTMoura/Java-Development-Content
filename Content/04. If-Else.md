# 04. If-Else

Less than:

```java
a<b;
```

Greater than:

```java
a>b;
```

Less than or equal:

```java
a<=b;
```

Greater than or equal to:

```java
a>=b;
```

Equal to:

```java
a==b;
```

Not Equal to:

```java
a!=b;
```


### Structure example

```java
if (20>18){
System.out.println("20 is greater than 18")
}
```
